## 介绍
###### 基于Spring Boot，JDK8，Swing组件开发，持续更新中...

## 安装执行
###### 1、下载源码，在根目录执行maven命令mvn clean package，可在target目录下生成jar包，然后直接CMD下执行命令java -jar xxx.jar即可。
###### 2、或者直接用IDE(Eclipse或IDEA)导入源码，找到TankBattleApplication类，执行其main方法即可。
